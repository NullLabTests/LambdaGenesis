## Overview

LambdaGenesis V2 transforms the original symbolic chemistry prototype into:

- graph-based chemistry
- executable symbolic structures
- spatial ecology
- catalytic interaction networks
- lineage tracking
- entropy analysis
- autocatalytic motif discovery

Inspired by:

- AlChemy
- Avida
- Tierra
- Lambda Calculus Chemistry
- Autocatalytic Set Theory
- Origin-of-Life Research

## Major Additions

### Graph Molecules
Molecules are executable graph structures.

### Spatial Ecology
Agents occupy a 2D world.

### Catalytic Networks
Reaction loops form persistent organizations.

### Entropy Metrics
Complexity and diversity are tracked over time.

### Evolutionary Lineages
All reactions generate ancestry graphs.

### Emergent Dynamics
Ecological pressure drives symbolic evolution.
